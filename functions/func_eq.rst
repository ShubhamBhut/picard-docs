.. Picard Function

$eq
===

| Usage: **$eq(x,y)**
| Category: conditional
| Implemented: Picard

**Description:**

Returns true if ``x`` equals ``y``.  Note that comparisons are case-sensitive.


**Example:**

The following statements will return the values indicated::

    $eq(,a)   ==>  ""
    $eq(a,)   ==>  ""
    $eq(a,A)  ==>  ""
    $eq(a,a)  ==>  "1"
