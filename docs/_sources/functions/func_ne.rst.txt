.. Picard Function

$ne
===

| Usage: **$ne(x,y)**
| Category: conditional

**Description:**

Returns true if ``x`` does not equal ``y``.  Note that comparisons are case-sensitive.


**Example:**

The following statements will return the values indicated::

    $ne(,a)   ==>  "1"
    $ne(a,)   ==>  "1"
    $ne(a,A)  ==>  "1"
    $ne(a,a)  ==>  ""
